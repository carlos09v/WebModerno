Asynchronous JavaScript And XML (AJAX)

  • É uma técnica usada com JavaScript
  • Atualiza a página (ou parte dela) sem recarregar (reload)
  • Comunica de forma assíncrona com um Servidor Web, ou seja, envia e recebe dados em background
  • Usa muito... Callback, Promise, Async/Await
    • Minhas anotações: >
    • Resumo: ( Técnica de requisição Assíncrona )
    • Formas de fazer Requisições:
      1. XMLHttpRequest (XHR) is a JavaScript API to create AJAX requests. Its methods provide the ability to send network requests between the browser and a server.
      2. Fetch API (do browser) (Mais moderno e simples) !
      3. Axios (biblioteca) (Ainda mais moderno e simples) !
      4. Jquery (Mais antiga (usa callback))
